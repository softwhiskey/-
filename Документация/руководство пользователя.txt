Руководство по эксплуатации бота.

Установка.

1.Скачать и распаковать архив (https://github.com/softwhiskey/-)
2.Файлы по адресу “prototypeBot\PrototypeTelegramBot\bin\Debug” распаковать в любую папку.
3.Зарегистрировать своего бота с помощью https://t.me/BotFather в Telegram.
4.Заменить токен, полученный в 3 пункте и данные от БД в файле config.json. (если используется локальная база данных, то достаточно заменить database (название БД), uid (логин) и password (пароль)
5.При необходимости импортировать таблицу applications в БД.
6.Добавить созданного бота в свой канал в Telegram.
7.Запустить файл PrototypeTelegramBot.exe

Использование и команды.

Для взаимодействия с ботом достаточно написать любую из нижеперечисленных команд:
1.   !help - список доступных команд
2.   !add - <имя> <телефон> <дата> (новые строки перечисляются через ;) - добавить новую запись
3.    !del <ID> - удалить запись с заданным идентификатором
4.    !sel <ID> - информация о записи с заданным идентификатором

Пример использования команд.

Добавление двух записей
!add Александр 79038765432 15/8/2022 15:32; Олег 79035769482 13/9/2021 15:32;
Удаление записи
!del 75
Информация о пользователе
!sel 25
